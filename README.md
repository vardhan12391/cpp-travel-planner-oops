# Travel Planning System (C++ OOP Project)

## Overview

The **Travel Planning System** is a console-based application developed using **C++** that demonstrates the practical implementation of **Object-Oriented Programming (OOP)** concepts.
The system allows users to manage travel-related information such as destinations, accommodation, budgets, and route planning in a structured and modular way.

This project is designed to showcase the use of **multi-file C++ programming and OOP design principles**.

---

## Features

* Destination management
* Accommodation planning
* Budget management
* Route planning between destinations
* Trip organization and management

---

## OOP Concepts Demonstrated

* Classes and Objects
* Encapsulation
* Modular Programming
* Header and Source File Separation
* Structured Code Design

## Getting Started

### 1. Clone the Repository

git clone https://github.com/vardhan12391/travel-planning-system.git

### 2. Navigate to the Project Directory

cd travel-planning-system

### 3. Compile the Program

g++ *.cpp -o travelplanner

### 4. Run the Program

./travelplanner
